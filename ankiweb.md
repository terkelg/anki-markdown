Write flashcards in Markdown with the same syntax highlighting you know from your code editor. Powered by [Shiki](https://shiki.style) (the engine behind VS Code), with 300+ languages, multiple themes, and light/dark mode support. Highlight specific [lines](https://github.com/terkelg/anki-markdown/blob/main/docs.md#line-highlighting), [words](https://github.com/terkelg/anki-markdown/blob/main/docs.md#word-highlighting), mark [errors and warnings](https://github.com/terkelg/anki-markdown/blob/main/docs.md#error-levels), use [focus mode](https://github.com/terkelg/anki-markdown/blob/main/docs.md#focus-mode) to dim surrounding code, and even highlight [inline code](https://github.com/terkelg/anki-markdown/blob/main/docs.md#inline-code-highlighting).

- **Editor-Grade Syntax Highlighting**: 300+ languages with accurate tokenization, not regex approximations
- **Advanced Code Annotations**: Line highlighting, word highlighting, focus mode, error/warning markers, and a copy button
- **Full Markdown**: Bold, italic, lists, blockquotes, tables, images, alerts, and more
- **Clean Card Design**: Polished light/dark styling that matches Anki's native UI
- **Mobile Compatible**: Works on AnkiMobile and AnkiDroid

See the [documentation](https://github.com/terkelg/anki-markdown/blob/main/docs.md) for all features and syntax guide.
